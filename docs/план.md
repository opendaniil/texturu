## Фаза 3 — Чанкинг + эмбеддинги + pgvector

Цель: подготовить RAG-слой независимо от Mastra.

- [ ] Модель данных:
  - [ ] `subtitle_chunks`:
    - `video_id`, `chunk_index`
    - `start_sec`, `end_sec`
    - `text`
  - [ ] `embeddings` (можно в той же таблице):
    - `embedding vector(<dim>)`
    - индекс: `ivfflat`/`hnsw` (что выберешь)
- [ ] Pipeline:
  - [ ] `chunk-subtitles` (30–60 секунд / или ~N токенов)
  - [ ] `embed-chunks` (батчами)
  - [ ] сохранить в pgvector
- [ ] API:
  - [ ] `GET /videos/:id/chunks` (для дебага, опционально)
  - [ ] `POST /videos/:id/search` (topK поиск по pgvector)
- [ ] Важно по стратегии:

  - делай вариант 1 (не лениво): эмбеддинги сразу после сабов — проще и стабильнее для демо/портфолио.
